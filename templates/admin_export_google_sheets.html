Лист 1: "Спортсмены по разрядам"

  • Все спортсмены, сгруппированные по разрядам
  • ФИО, дата рождения, пол, школа
  • Список турниров, в которых участвовал спортсмен
  • Количество участий (всего и бесплатных)
  • Цветовая подсветка для спортсменов с бесплатными участиями

Лист 2: "Анализ по школам"

  • Все школы с их спортсменами
  • Статистика по каждой школе: всего детей, участий, бесплатно/платно
  • Группировка по разрядам внутри каждой школы (с отдельными заголовками)
  • Для каждого спортсмена: турниры, количество участий, бесплатные участия
  • Цветовая подсветка (зелёный фон) для спортсменов с бесплатными участиями
🎨 Точечное форматирование
  • [БЕСПЛАТНО] текст перед турнирами - отмечены бесплатные турниры
  • 4+ 2-3 1 - количество бесплатных участий (разные оттенки зелёного)
  • Зелёный фон + зелёный текст - ячейка с турнирами, если есть бесплатные
  • Колонка "Пол" - удалена (пол указан в разряде)
📖 Инструкция по настройке Google Sheets API
Шаг 1: Создать проект в Google Cloud Console
  1. Откройте Google Cloud Console
  2. Создайте новый проект
  3. Включите Google Sheets API и Google Drive API
Шаг 2: Создать Service Account
  1. Перейдите в "IAM и администрирование" → "Сервисные аккаунты"
  2. Создайте новый сервисный аккаунт
  3. Дайте ему роль "Редактор" или "Владелец"
Шаг 3: Скачать ключ
  1. Откройте созданный Service Account
  2. Перейдите на вкладку "Ключи"
  3. Создайте новый ключ (тип JSON)
  4. Файл автоматически скачается
Шаг 4: Загрузить ключ
  1. Переименуйте скачанный файл в google_credentials.json
  2. Загрузите его в корень проекта
Шаг 5: Настроить доступ к таблице
  1. Создайте Google Таблицу
  2. Поделитесь ей с email Service Account (из шага 2)
  3. Дайте права "Редактор"
