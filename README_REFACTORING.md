## 📋 Что нужно сделать перед запуском

### 1. Создать файл .env

Скопируйте `ENV_EXAMPLE.txt` в `.env` и заполните:

```bash
# Обязательные переменные
SECRET_KEY=<сгенерируйте через: python -c "import secrets; print(secrets.token_hex(32))">
ADMIN_USERNAME=admin
ADMIN_PASSWORD_HASH=<сгенерируйте через: python scripts/generate_password_hash.py ваш_пароль>

# Остальные переменные (опционально)
DATABASE_URL=sqlite:///figure_skating.db
UPLOAD_FOLDER=uploads
LOG_LEVEL=INFO
FLASK_DEBUG=False
```

### 2. Сгенерировать хеш пароля

```bash
python scripts/generate_password_hash.py ваш_пароль
```

Скопируйте полученный хеш в `.env` как `ADMIN_PASSWORD_HASH`.

### 3. Проверить что все работает

```bash
python app.py
```
